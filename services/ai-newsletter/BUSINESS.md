# lutra AI 新闻简报服务

## 愿景
每日自动收集、筛选、总结 AI/科技领域的最新动态，生成深度分析文章，通过多平台分发建立影响力，最终实现内容变现。

## 商业模式

### 三条收入路径

#### 1. 免费内容引流 (流量)
- **多平台发布**: 每日文章发布到 Medium、公众号、知乎、掘金
- **SEO 优化**: 抢占「AI 新闻」关键词
- **目标**: 10,000+ 月活读者

#### 2. 订阅服务 (收入)
| 层级 | 价格 | 内容 |
|------|------|------|
| 免费 | $0 | 每日简报 + 免费文章 |
| 基础 | ¥19/月 | 每日深度分析 + 历史存档 |
| 专业 | ¥49/月 | API 访问 + Slack 推送 + 定制分析 |

#### 3. 内容变现 (品牌)
- **付费专栏**: 深度系列文章 ($5-10/篇)
- **企业内训**: AI 趋势解读服务
- **咨询顾问**: AI 选型建议

### 目标用户
- 投资者（了解 AI 趋势）
- 开发者（跟进技术动态）
- 产品经理（竞品分析）
- 科技爱好者

## 技术架构

```
数据采集层
├── Hacker News API
├── Twitter/X API (免费端)
├── Reddit API
└── AI 垂直媒体 RSS

处理层
├── 新闻去重
├── 关键词过滤 (AI, LLM, Agent...)
├── 重要性评分
└── 摘要生成 (本地模型 / API)

分发层
├── 邮件发送 (SendGrid / Resend)
├── Webhook (Slack, Discord)
└── API (供专业用户)

存储层
├── SQLite (新闻缓存)
├── LanceDB (向量相似搜索)
└── 文件系统 (JSON)
```

## 竞争优势

1. **全自动化** - 每天自动生成文章，人工成本为 0
2. **AI 原生** - 用 AI 分析 AI 新闻，差异化定位
3. **多平台分发** - 一次生成，多平台发布
4. **中英双语** - 覆盖中文 + 英文读者
