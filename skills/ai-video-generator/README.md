# AI Video Generator

Unified AI video generation interface for anime and cartoon production.

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Configure API keys
export RUNWAY_API_KEY="your-runway-key"

# Generate video from text
python scripts/generate.py text "anime girl walking" --duration 5
```

## Features

- **Multi-provider support**: Runway, Pika, Kling, Luma
- **Text to video**: Prompt-based generation
- **Image to video**: Animate static images
- **Video extension**: Extend existing clips
- **Motion presets**: Pre-configured motion styles
- **Batch processing**: Generate multiple scenes
