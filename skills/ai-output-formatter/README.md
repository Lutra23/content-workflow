# Output Formatter

Multi-format video export.

## Usage

```bash
# Export for platform
python scripts/generate.py export video.mp4 output.mp4 --platform youtube

# Compress
python scripts/generate.py compress video.mp4 small.mp4

# Generate GIF
python scripts/generate.py gif video.mp4 preview.gif

# Show info
python scripts/generate.py info video.mp4
```
